# HORROLIVE 拡張仕様 v3.2

## 追加盤面：雨守市立・黒羽分校

廃病院に加え、深夜の廃校を選択可能な独立盤面として追加する。昇降口、二階教室棟、旧放送室、階段踊り場を進み、出席簿・録音・写真を照合して下校放送の正体を記録する。

Chapter 2 では「放送委員会・最終録音テープ」を回収せずに先へ進むと、2年B組前へ戻される。Main は同じ廊下を映す一方、PIP はループ直前の映像を短時間保持し、コメントだけが先に周回を指摘する。テープ回収後にのみ旧放送室へ進める。

## ホラー演出の原則

恐怖の中心は怪異そのものではなく、Main・PIP・Chat の観測不一致に置く。

- Main：空の廊下や教室を映し、異常を断定しない。
- PIP：遅延、短いフリーズ、反復の後に、髪・肩・指先など部分的な輪郭だけを残す。
- Chat：画面より先に異変を予告し、発生後には沈黙や距離の食い違いで余韻を作る。

怪異は段階的に近づけ、予兆から記録可能時間、余韻までを一つのイベントとして扱う。通常の異変は通知音、囁き、画面のズレで示し、強いスティンガーは終盤の山場に限定する。

## 周回・メタ進行

盤面ごとに挑戦回数、到達エンディング、収集物、記録済み怪異を保存する。初回クリア後は周回モード「深夜再送 / DEEP BROADCAST」を解放する。

深夜再送は同接 2,370 から開始し、初期電池を減らし、電池消費を速め、怪異の予兆と記録可能時間を短縮する。盤面ごとの全記録や特殊条件達成を残し、別盤面の進行とは混在させない。

## 抑制ルール

- 流血、肉体損壊、怪異の全身・正面顔を常用しない。
- 頻繁なジャンプスケアや大音量で恐怖を消費しない。
- 戦闘、武器、明るいゲーム的 UI を主役にしない。
- 滑らかすぎる怪異アニメーションを避け、欠落フレームや位置の飛びを使う。
- 怪異を説明し切らず、どの画面が正しいか判断できない余白を残す。
