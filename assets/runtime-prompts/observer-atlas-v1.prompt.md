# Observer atlas v1 generation prompt

Reference: `assets/concepts/observer-direction-v4-v2.png`

Create a production-ready 2D pixel-art horror entity atlas derived faithfully from the approved Observer direction sheet.

- One landscape PNG, exactly 3 columns x 1 row, equal cells with generous padding.
- Perfectly flat chroma magenta `#FF00FF` background. No dividers, labels, UI, camera frame, focus box, scanlines, monitor, scene, curtain, player, flashlight, floor, shadow, blood, or particles.
- The same entity in every cell: an extremely slender adult human-like figure, shoulder-length tangled near-black hair, desaturated gray-green institutional gown/apron, too-long arms, still posture, and a perfectly smooth blank oval face. No eyes, mouth, nose holes, wounds, symbols, or glow.

Cell order:

1. Far tier, full body front-facing with arms hanging.
2. Mid tier, full body in a slight 3/4 turn with one shoulder forward.
3. Near tier, contained waist-up / head-and-shoulders view with the blank face readable.

Use crisp authored low-detail pixel art in a low-contrast charcoal / gray-green palette with dim neutral lighting. No fantasy-monster, zombie, nun, child, or gore cues. Use hard edges suitable for chroma-key cleanup and keep the key color off the entity.
